# Student Attendance System

[![CI Pipeline](https://github.com/sathsaranikawindy/STUDENT-ATENDENCE-SYSTEM/actions/workflows/ci.yml/badge.svg)](https://github.com/sathsaranikawindy/STUDENT-ATENDENCE-SYSTEM/actions/workflows/ci.yml)

## Project Description
This project is a simple Student Attendance Management System built using HTML, CSS, JavaScript, and Node.js with MongoDB integration.

## Features
- Add student attendance
- Delete records
- Update attendance table
- MongoDB database connection
- CI pipeline using GitHub Actions

## CI/CD Pipeline
This project uses **GitHub Actions** for Continuous Integration.

Pipeline steps:
1. Checkout repository
2. Setup Node.js
3. Install dependencies
4. Start server test
5. Verify MongoDB connection
6. Complete pipeline

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- MongoDB
- GitHub Actions
